# Visita Virtual - Contenido Multimedia

Esta carpeta contiene las imágenes y videos de la visita virtual del museo.

## Estructura de archivos

Coloca aquí:
- **Imágenes**: JPG, PNG, WEBP
- **Videos**: MP4, WEBM

## Nombres de archivo recomendados

Para mejor organización:
- `sala-01.jpg`, `sala-02.jpg`, etc.
- `video-tour-01.mp4`, `video-tour-02.mp4`, etc.

El sistema detectará automáticamente el tipo de archivo (imagen o video) y lo mostrará apropiadamente.
